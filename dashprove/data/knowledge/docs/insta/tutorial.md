# Documentation

Insta is a snapshot testing tool for Rust. It consists of multiple components, all of which you can
find documented here.

* [Getting started][15] for a short introduction to insta.
* [Snapshot types][16] for the different types of snapshots.
* [Serializers][17] to learn about the different serializer choices.
* [Snapshot files][18] documents the snapshot file format.
* [Redactions][19] to learn about the snapshot redactions feature.
* [Filters][20] to learn about the snapshot string filtering feature.
* [Advanced Features][21] to learn about more advanced features such as globbing.
* [Testing CLIs][22] to learn how to use `insta-cmd` to test CLIs.
* [Settings][23] for how you can customize insta's behavior.
* [Patterns][24] common patterns for using insta.
* [Cargo Insta][25] for how to use the companion tool.
* [VS Code Extension][26] if you want to use insta with Visual Studio Code.

In addition have a look at the in-depth [API documentation][27] which covers the entire API of the
crate.

Don't like reading? There is a [12 minute introduction screencast][28] that walks you through how it
works.

Found an issue? You can [edit this page][29] on GitHub.
