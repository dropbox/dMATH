## KLEE Symbolic Execution Engine

KLEE is a dynamic symbolic execution engine built on top of the [LLVM][1] compiler infrastructure,
and available under the UIUC open source license. For more information on what KLEE is and what it
can do, see the [OSDI 2008][2] paper.
