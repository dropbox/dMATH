# [Fuzzing with afl.rs][1]

[American fuzzy lop][2] (AFL) is a popular, effective, and modern fuzz testing tool. [afl.rs][3]
allows one to run AFL on code written in [the Rust programming language][4].

[2]: http://lcamtuf.coredump.cx/afl/
[3]: https://github.com/rust-fuzz/afl.rs
[4]: https://www.rust-lang.org
