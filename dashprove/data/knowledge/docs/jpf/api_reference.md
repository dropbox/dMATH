# Java Pathfinder

This is the main page for Java™ Pathfinder (JPF). JPF is an extensible software model checking
framework for Java™ bytecode programs. The system was developed at the [NASA Ames Research
Center][25], open sourced in 2005, and is freely available on this server under the [Apache-2.0
license][26].

* [Introduction][27] - a brief introduction into model checking and JPF
* [How to obtain and install JPF][28] - everything to get it running on your machine
* [How to use JPF][29] - the user manual for JPF
* [Developer guide][30] - what's under the hood
* [JPF core project][31] - description and link to jpf-core
* [Projects][32] - link to currently active and legacy extensions of JPF
* [Workshops][33] - link to annual workshops on JPF
* [Google Summer of Code][34] - link to JPF's participation in the Google Summer of Code program,
  current ideas page [here][35].
* [Mailing list][36] - discussion about JPF takes place in Google Groups [JPF Google Group][37]
* [Discord][38] - join our [Discord server][39] for more discussions
