# cargo-vet

[[crates.io]][1] [[Rust CI]][2]

The `cargo vet` subcommand is a tool to help projects ensure that third-party Rust dependencies have
been audited by a trusted entity. It strives to be lightweight and easy to integrate.

More details available in the [book][3].
