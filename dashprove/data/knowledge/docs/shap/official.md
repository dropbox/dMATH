# Welcome to the SHAP documentation[][2]

[[_images/shap_header.png] ][3]

**SHAP (SHapley Additive exPlanations)** is a game theoretic approach to explain the output of any
machine learning model. It connects optimal credit allocation with local explanations using the
classic Shapley values from game theory and their related extensions (see [papers][4] for details
and citations).

## Install[][5]

SHAP can be installed from either [PyPI][6] or [conda-forge][7]:

pip install shap
or
conda install -c conda-forge shap
