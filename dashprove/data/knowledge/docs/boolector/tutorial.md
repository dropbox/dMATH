# Welcome to Boolector’s API documentation![¶][1]

## Introduction[¶][2]

> [Boolector][3] is an SMT solver for the theories of bit-vectors, arrays, uninterpreted functions
> and their combinations. Boolector supports [BTOR][4], [SMT-LIB v1][5], and [SMT-LIB v2][6] as
> input format and can be either used as a stand-alone SMT solver, or as back end for other tools
> via its public API. This is the documentation of Boolector’s public **C interface** and **Python
> interface**. For further information and the latest version of Boolector, please refer to
> [https://boolector.github.io][7].
